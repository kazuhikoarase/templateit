templateit

Usage

1. Place templateit files under your project directory.

project_name/
             templateit/
                        README.txt
                        build.xml
                        .gitignore
                        templateit.jar

2. Run build.xml

3. Check your template jar (executive)

project_name/
             templateit/
                        dist/
                             project_name + '.v' + timestamp + '.jar'
