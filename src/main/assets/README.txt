templateit

Usage

1. Place templateit files under your project directory.

your_project/
             templateit/
                        README.txt
                        build.xml
                        .gitignore
                        templateit.jar

2. Run build.xml.

your_project/
             templateit/
                        build.xml <-

3. Check your template jar (executable).

your_project/
             templateit/
                        dist/
                             your_project + '.v' + timestamp + '.jar'
